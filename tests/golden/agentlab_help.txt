Usage: agentlab [OPTIONS] COMMAND [ARGS]...

  AgentLab VNextCC — agent optimization platform.

  A product-grade platform for iterating ADK agent quality. CLI-first, API-
  ready, with a web console for visual insight.

Options:
  --version    Show the version and exit.
  --classic    Launch the classic REPL shell instead of the new Workbench UI.
  --quiet      Suppress the branded startup banner.
  --no-banner  Suppress the branded startup banner.
  --help       Show this message and exit.
