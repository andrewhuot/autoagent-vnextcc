## loop
EXIT=0
Usage: autoagent loop [OPTIONS]

  Run the continuous autoresearch loop.

  Observes agent health, proposes improvements, evaluates them, and deploys
  accepted changes — automatically, cycle after cycle.

  Examples:   autoagent loop   autoagent loop --max-cycles 100 --stop-on-
  plateau

Options:
  --max-cycles INTEGER            Maximum optimization cycles.  [default: 50]
  --stop-on-plateau               Stop if no improvement for 5 consecutive
                                  cycles.
  --delay FLOAT                   Seconds between cycles.  [default: 1.0]
  --schedule [continuous|interval|cron]
                                  Scheduling mode. Defaults to autoagent.yaml
                                  loop.schedule_mode.
  --interval-minutes FLOAT        Interval minutes for --schedule interval.
  --cron TEXT                     Cron expression for --schedule cron (5-field
                                  UTC).
  --checkpoint-file TEXT          Checkpoint file path. Defaults to
                                  autoagent.yaml loop.checkpoint_path.
  --resume / --no-resume          Resume from checkpoint when available.
                                  [default: resume]
  --full-auto                     Danger mode: auto-promote accepted configs
                                  and skip manual gates.
  --db TEXT                       Conversation store DB.  [default:
                                  conversations.db]
  --configs-dir TEXT              Configs directory.  [default: configs]
  --memory-db TEXT                Optimizer memory DB.  [default:
                                  optimizer_memory.db]
  --help                          Show this message and exit.

## mcp-server
EXIT=0
Usage: autoagent mcp-server [OPTIONS]

  Start MCP server for AI coding tool integration.

  Runs in stdio mode by default for Claude Code, Cursor, Codex, and other MCP-
  compatible tools. Supply --port to serve streamable HTTP locally.

  Setup for Claude Code (project-scoped .mcp.json example):   {
  "mcpServers": {       "autoagent": {         "command": "python3",
  "args": ["-m", "mcp_server"]       }     }   }

  Examples:   autoagent mcp-server                         # Stdio mode
  (default)   autoagent mcp-server --host 127.0.0.1 --port 8081

Options:
  --host TEXT     HTTP bind host when using --port.  [default: 127.0.0.1]
  --port INTEGER  Start streamable HTTP mode on this port. Defaults to stdio
                  mode.
  --help          Show this message and exit.
